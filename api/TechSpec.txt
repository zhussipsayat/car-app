.NET Backend разработчик
Тестовое задание
Реализовать Web API, который должен удовлетворять следующим требованиям:

+1. Структура БД должна создаваться с помощью миграций EF:

	1. Структура БД должна состоять из 2 таблиц (кроме системных Asp):*
		+1. car (id - первичный ключ, h_color_id вторичный ключ на таблицу h_color, brand_name строковый, model_name строковый)
		+2. h_color (id первичный ключ, name строковый)
	2. В миграциях также должно быть предусмотрено автоматическое заполнение тестовыми данными:
		+1. Car - 2 тестовые записи
		+2. H_color - 5 тестовых записей (цветов)
		+3. Системные таблицы (Asp)*

+2. Требования к АРІ:
	1. В АРІ должен быть предусмотрен 1 Controller и 5 Actions:
		+HTTP GET - HColor получение списка всех цветов (доступ с авторизацией)
		+HTTP GET - Car получение списка всех автомобилей (доступ с авторизацией)
		+HTTP POST - Car добавление нового автомобиля (доступ с авторизацией)		
		+HTTP PUT - Car изменение данных об автомобиле (доступ с авторизацией)
		+HTTP DELETE Car удаление автомобиля (доступ с авторизацией)
	+2. Способ авторизации JWT
	+3. Для взаимодействия с БД использовать EF Core
+3. К WebApi должен быть подключен Swagger UI.
